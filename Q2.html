There are different types of Operators

  • 1. Arithmetic Operators :

    Arithmetic operators are used to do mathematical operations like addition, subtraction, multiplication, division, etc. It simply takes numerical values as operands, performs an arithmetic operation, and returns a numerical value.

    They are Addition ( + ), Subtraction ( - ), Multiplication (*), Division (/), Modulus(%), Increment and Decrement(post & pre)

    let num1 =100;
    let num2 =2;

    • let result = num1 + num2; // (Answer will be 102) // Addition Operator
    • let result = num1 - num2; // (Answer will be 98) // Subtraction Operator
    • let result = num1 * num2; // (Answer will be 200) // Multiplication Operator
    • let result = num2 / num1; // (Answer will be 50) // Division Operator
    • let result = num2 % num1; // (Answer will be 0) // Modulus Operator

    • let x = 3; const y = x++; (Answer will be x:4, y:3) // Pre-increment Operator. If used postfix, with operator after operand (for example, x++), the increment operator increments and returns the value before incrementing.

    • let x = 3; const y = ++x; // (Answer will be x:4, y:4) // Post-increment Operator. If used prefix, with operator before operand (for example, ++x), the increment operator increments and returns the value after incrementing.

    • let x = 3; const y = x--; (Answer will be x:2, y:3) // Pre-decrement Operator. If used postfix, with operator after operand (for example, x--), the decrement operator decrements and returns the value before decrementing.

    • let x = 3; const y = --x; // (Answer will be x:2, y:2) // Post-decrement Operator. If used prefix, with operator before operand (for example, --x), the decrement operator decrements and returns the value after decrementing.

  • 2. Assignment Operator

    Assignment operators do something with the value on the right side to set the variable on the left side or we can say Assignment operators are an essential aspect of JavaScript that allow you to assign values to variables and perform arithmetic operations in a concise manner. We can divided into two types:

    • Simple Assignment(=):The simple assignment operator is used to assign a single value to a variable. It takes the value on the right hand side (RHS) and assigns it to the variable on the left-hand side (LHS). Example:
      let length =50;
      In the above example, we used the simple assignment operator(=) to assign the value 50 to the variable 'length'.

    • Compound Assignment:Compound assignment operators combine arithmetic operations with assignment. These operators perform the specified arithmetic operation on the variable's current value and then assign the result back to the variable. They many shorthand way of writing expressions.

    • += Addition and assignment
      let x = 6;
      x += 4; // Equivalent to x = x + 4;
      Output: 10
      In the example above, the += operator adds 4 to the current value of x (which is 6) and assign the result (10) back to x.
    • -= Subtraction and assignment
      the -= operator subtracts from the current value and assigns the result.
    • *= Multiplication and assignment
      the *= operator multiplies the current value and assigns the result
    • /= Division and assignment
      /= operator divides the current value and assigns the result
    • %= Modulus and assignment (remainder after division)
      %= operator finds the remainder of dividing the current value of the dividend and assigns the result
    • **= Exponentiation and assignment
      **= operator calculates the value of the current base raised to the power and assigns the result.

  • 3. Comparison Operator

    Comparison Operators are used to compare values and return Boolean results ( true or false) based on the comparison. There are four types of comparison operators.

    • Equality (==) and Strict Equality (===) : The equality operator (==) and strict equality operator (===) are used to compare two values for equality. The equality operator (==) compares both the values. It returns true if both values are equal. The strict equality operator (===) compares both the values and their types. It only returns true if both values and types are the same.
    • Inequality (!=) and Strict Inequality (!==) : The inequality operator (!=) compares two values to check if they are not equal and return a Boolean value. But in the case of Strict Inequality operator (!==) checks both the values and their data types. It returns true only if both the values and the types are different
    • Greater than (>), Less than (<), Greater than or equal to (>=), Less than or equal to (<=) : The greater than operator (>) checks if the left operand is numerically greater than the right operand. The less than operator (<) checks if the left operand is numerically less than the right operand. The greater than or equal to operator (>=) checks if the left operand is numerically greater than or equal to the right operand. The less than or equal to operator (<=) checks if the left operand is numerically less than or equal to the right operand.
    • . Nullish Coalescing (??) : The nullish coalescing operator (??) is used to handle null or undefined values. It returns the right-hand operand if the left-hand operand is null or undefined; otherwise, it returns the left-hand operand.

  • 4. Logical Operator

    Logical operators perform logical operations and return a boolean value, either true or false. There are three main logical operators AND (&&), OR (||), and NOT (!).

    • AND Operator(&&) :

      This operator returns true if both operands are true, and false otherwise. It is a logical first operator, meaning that if the first operand is false, the second operand is not evaluated.

    • Logical OR(||) Operator :

      This operator returns true if at least one of the operan's is true, an' false otherwise. Like && operator, it is also a short-circuit operator.

    • Logical Not (!)

      This operator inverts the Boolean value of the operand. If the operand is true, the operator returns false, and if the operand is false, the operator returns true. These logical operators can be used to perform a variety of tasks, such as testing multiple conditions or combining multiple conditions in a single expression. They are often used with other techniques and operations to perform more complex tasks, such as flow control and data validation. We will be looking at their applications in further lectures.

  • 5. Unary Operator

    Unary operators are operators that perform actions on a single operand. Let's explore some of the Unary operators:

    • Unary Plus('+') :

      The unary plus operator is used to convert its operand into a number. While this might seem redundant, it becomes valuable when dealing with user inputs that need to be explicitly converted to numbers.
      Example:
      let numString = "10";
      let negativeNum = +numString; // Output: 10
      We are declaring a variable numString and assigning it the value "42", which is a string representing the characters "4" and "2".
      Then, We use the unary plus operator (+) on numString to convert it into a number. This unary plus operation constrain the string "42" into the numeric value 42

    • Unary negation('-')

      The unary negation operator is used to negate the value of its operand, effectively changing its sign. Example:
      let num = 10;
      let negativeNum = -num; // Output: -10
      The unary negation operator simply changes the sign of a number. If the number is positive, it becomes negative, and if it's negative, it becomes positive

    • Logical NOT('!') :

      The logical NOT operator is used to negate the Boolean value of its operand. It converts true to false and vice versa. It is a unary operator.We have already known the 'Logical NOT' operator.

    • Increment (++) and Decrement (--) Operators :

      The increment (++) and decrement (--) operators are used to increase or decrease the value of a variable by 1, respectively.

    • typeof Operator : The typeof operator is used to determine the data type of its operand or we can say it allows checking the data type of the variable.
    • void operator : The void operator is used to evaluate an expression and return undefined. In JavaScript, the void operator is used to explicitly return undefined. It's a unary operator, meaning only one operand can be used with it. You can use it like shown below — standalone or with a parenthesis.
      void expression
      void (expression)
    • delete Operator :

      The delete operator is used to delete an object's property or an element of an array.
      let user = {name:"Raj", age:25, Gender:"Male"}
      delete user.age; // Deletes the age property.
      console.log(user); // Output: {name:"Raj", Gender:"Male"}

  • 6. Ternary Operator

    The ternary operator is a shorthand way to write simple conditional statements. The syntax is in the following way:

    Condition ? expression 1 : expression 2

    Condition A boolean expression that is evaluated.
    expr1 The value to return if the condition is true.
    expr2 The value to r!turn if the condition is false.

    const age = 18;
    const canVote = age >= 18 ? ' YES' : 'NO' ;
    // OUTPUT : 'YES'

  • 7. String Operator (string concatenation +)

    String operators in JavaScript are used to concatenate strings< The syntax is in the following way: str1 + str2

    str1 The first string to concatenate.
    str2 The second string to concatenate.

    let first_Name ='Raj';
    let last_Name ='Kumar';
    let full_Name = first_Name + ' ' + last_Name;
    // Output : Raj Kumar;
